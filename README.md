# ИТОГОВЫЙ ВЫВОД:
Генераторы и итераторы Python представляют собой инструменты, используемые для обработки данных. Основное назначение итераторов - это упрощение навигации по элементам объектов, которые могут возглавить собой некоторую коллекцию, например список, словарь и т.п., а генераторы позволяют значительно упростить работу по построению итераторов.
